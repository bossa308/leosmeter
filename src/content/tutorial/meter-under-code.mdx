## UNDER คืออะไร?

**UNDER** เป็น Error Code ที่แสดงขึ้นเมื่อค่าที่วัดได้ **ต่ำกว่าขีดจำกัดล่าง (Under Range)** ของช่วงการวัดที่ตั้งค่าไว้ ซึ่งอาจเกิดจาก:

- สัญญาณ Input ต่ำกว่า 0% ของ Range (เช่น กระแส < 4mA สำหรับ 4-20mA Input)
- CT ไม่ได้ต่อ หรือ CT ต่อขั้วผิด
- สายสัญญาณขาด หรือหัว Terminal หลวม

## การตรวจสอบทีละขั้นตอน

### ขั้นที่ 1: ตรวจสอบ Input Signal

ใช้มัลติมิเตอร์วัดค่าสัญญาณที่ขั้ว Input ของมิเตอร์

| สัญญาณ | ค่าปกติ | ค่าที่จะเกิด UNDER |
|--------|---------|-------------------|
| 4-20 mA | 4–20 mA | < 3.8 mA |
| 0-10 V | 0–10 V | < -0.5 V |
| CT | ตามสเปค | ไม่มีกระแส |

### ขั้นที่ 2: ตรวจสอบ Wiring

- ตรวจดูว่าสาย + และ − ต่อถูกขั้ว
- ขันหัว Terminal ให้แน่น
- ตรวจสอบว่า CT Loop ปิดครบ

### ขั้นที่ 3: ตรวจสอบ Scaling

เข้าเมนูตั้งค่าของมิเตอร์ ตรวจสอบว่า **Input Range** และ **Display Range** ตั้งค่าถูกต้อง

## สรุป

| สาเหตุ | วิธีแก้ |
|--------|---------|
| สาย Input หลุด | เสียบสายใหม่ให้แน่น |
| CT ไม่ครบ Loop | ปิด Loop CT ให้ครบ |
| ตั้ง Range ผิด | แก้ค่า Input Range ในเมนู |
| Sensor เสีย | เปลี่ยน Sensor |
